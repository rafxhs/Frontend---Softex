Aula 1 - Introdução ao HTML

Conhecendo o HTML e sua estrutura básica

  1. O que é o HTML e qual é o seu papel na construção de páginas web? Descreva brevemente a estrutura básica de um documento HTML.

    • O HTML é uma linguagem de marcação de hipertexto utilizada para construir uma página web, seu principal papel é a estruturação de conteúdo. O documento HTML é composto por a declaração do tipo de documento com o !DOCTYPE html, e depois a abertura da tag html e dentro da própria tag, o atributo lang = "" e dentro dos parênteses o idioma utilizado. Por fim, mas que é de grande importância dentro da tag html, há a tag head e body, a tag head responsável por ter dentro dela metadados e codificação de caracteres e a body toda a parte vísivel da página.

  2. Como o HTML se relaciona com outras tecnologias da web, como o CSS e o JavaScript? Como essas três tecnologias trabalham em conjunto para criar páginas web interativas e estilizadas?

    • A relação entre essas três tecnologias ocorre pela possibilidade de usar a linguagem JavaScript dentro do HTML utilizando a tag script ou linkar um arquivo JS através também da tag script, só que utilizando o atributo src ="" e colocar dentro desse atributo o caminho para o arquivo JS. Enquanto que, com o CSS é possível utilizá-lo de forma inline, interno (através da tag style) ou externo (por meio da tag link).
      Essas três tecnologias são de grande importância para construção de uma página web, seus trabalhos são impregados da seguinte maneira: o HTML é responsável por toda a estruturação de conteúdo, desde textos, imagens, formulários, etc. Já o CSS é responsabilizado por do layout e design de uma página, utilizando através de suas regras os seletores (de tipo, classe ou id) e deles colocando suas propriedades e seus respectivos valores. Tendo todo o conteúdo e o seu design pronto, o JavaScript é encarregado por adicionar interatividade aos elementos da página, como também possibilitar a manipulação do DOM (Document Object Model).

Vinculando páginas, arquivos e âncoras

  1. Como a tag a é utilizada para criar links em HTML? Explique como você pode vincular para outras páginas, para arquivos locais e criar âncoras dentro de uma página.

    • A tag a criar links usando o atributo href = "" dentro da sua tag de abertura e dentro da própria tag estará algum texto que levará o usuário para o caminho desejado. O vínculo para outras páginas ocorre ao utilizar o atributo href = "aqui será colocado a URL de algum site"dentro de sua abertura e dentro da própria tag terá algum texto. Já a segunda maneira de vincular é para arquivos locais, colocando dentro dentro de seu atributo href = "" o diretório ou subdiretório que estará com algum tipo de arquivo. E por último, há também a criação de âncoras dentro da página, que é feito por meio da definção de um ID em um elemento de destino e usa o href = "" para referenciar esse id.

  2. Além do atributo href, quais outros atributos importantes a tag pode ter? Como o atributo target pode ser usado para controlar como o link é aberto pelo navegador?

    • Há também outros atributos como, target que define onde o link será aberto, download indica que o link é um download, rel utilizado para marcar um link de saída dentro de um tipo específico de conteúdo, etc. O atributo target possui seus valores que serão a chave para o controle de como o link será aberto, há o target="_blank" que serve para abrir o link numa nova aba no navegador, o target="_blank" abre o link na mesma aba ou janela em que o link foi clicado, o target="_blank" abre o link na janela inteira e remove qualquer iframe ou frame, entres outros tipos de valores para o atributo target que possibilitam o controle de onde será a abertura dos links pelo browser.

Incorporando imagens e mídias

  1. Descreva a diferença entre os atributos src e alt em relação à tag img. Por que é importante fornecer um atributo alt adequado para imagens?

    • O atributo src (source) é obrigatório para tag img pois é usado para colocar o caminho do arquivo da imagem, já o atributo alt serve para definir um texto que descreve a imagem. É importante fornercer o atributo alt de forma adequada, pois torna a página inclusiva e acessível para pessoas com problemas visuais ou até mesmo em caso do qual a imagem não é carregada, tendo uma descrição adequada faz com que seja mantindo por parte do usuário a compreensão do conteúdo da página.

  2. Além de imagens, quais outros tipos de mídia podem ser incorporados em uma página HTML? Como você pode garantir que a mídia incorporada seja responsiva e se ajuste bem em diferentes dispositivos?

    • Além da tag img que possibilita colocar imagens num documento HTML, há também as tags de multimídia video, audio e figure. Para garantir que a mídia incorporada seja responsiva, é necessário usufruir de técnicas de responsividade (ex.: viewports, media queries, unidades relativas ou flebox) para tornar não só imagens responsivas como qualquer outro tipo de mídia.

Criando tabelas básicas

  1. Quais são as principais tags usadas para criar tabelas em HTML? Explique a diferença entre as tags th e td, e quando você as usaria.

    • As principais tags para construir uma tabela são: table, th, tr e td. Apesar de ambas as tags th e td
      • serem utilizadas para criar células em uma tabela, a diferença entre as duas é que a tag th é usada para definir células do cabeçalho da tabela, já o td trata-se da célula de dados de uma tabela. Caso eu fosse construir uma tabela, após ter definido uma tabela com a tag table, dentro do tr eu usaria o th colocar todas as células dessa linha do tipo título, e logo após, as demais seriam dados e utilizaria a tag td.
  2. Embora tabelas sejam usadas para exibir dados tabulares, às vezes é recomendado evitá-las em favor de outras formas de apresentação de dados. Por quê? E quais são os cenários em que as tabelas são apropriadas?

    • Sabendo que o uso das tabelas são melhores para dados tabulares, é apropriado utilizar tabelas para representar dados de um banco de dados, já em alguns casos como, ter um site que seja responsivo não é indicado utilizar tabelas, pois esses dados não são tabulares e tornar uma tabela responsiva para todos os tipos de dispositivos é algo mais difícil (principalmente para dispositivos móveis).

Técnicas avançadas de estruturação com elementos HTML5 semânticos

  1. O que são elementos HTML5 semânticos e por que são importantes na estruturação de uma página? Cite três exemplos de elementos semânticos e explique quando você os usaria.

    • A utilização de elementos semânticos do HTML5 (ex.: header, nav, main, footer, etc.) durante a criação de um documento HTML deixa o código bem estruturado e o torna de fácil legibilidade para um desenvolvedor. Além disso, usar elementos semânticos possibilita os algoritmos de indexação em sites de busca e as tecnologias assistivas a uma melhor compreensão do contexto e conteúdo do seu site entregando uma melhor experiência para os usuários.

  2. Além da semântica, como os elementos HTML5 podem afetar a acessibilidade das páginas web? Por que é fundamental considerar a acessibilidade ao escolher elementos de estruturação?

    • Os elementos semânticos impactam a acessibilidade de forma positiva, pois é através dos elementos semânticos que um website torna-se de melhor compreensão por sua estrutura ficar mais organizada e apresentar clareza dos conteúdos expostos, ajudando as tecnologias assistivas a interpretar todo o conteúdo e oferecer uma navegação mais eficiente. É fundamental considerar a acessibilidade ao escolher elementos semânticos, devido a entrega de melhoria da experiência do usuário, inclusão, melhor manutenção do código e melhoria na indexação.
